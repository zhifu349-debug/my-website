## 1. 系统概述

### 这是什么？

这是一个用来**管理网站内容**的工具，就像用 Word 写文章一样简单。你不需要懂编程，不需要懂技术，只要会打字、会点鼠标就能使用。

### 你需要什么？

- ✅ 一台电脑（Windows、Mac 都可以）
- ✅ 一个浏览器（推荐 Chrome 或 Edge）
- ✅ 会打字的基本操作

**就这么简单！**

### 内容类型说明

| 图标 | 类型 | 什么时候用 | 例子 |
|------|------|-----------|------|
| ⭐ 推荐文章 | 想推荐多个产品时 | "2026年最好的5个VPS主机" |
| 📊 产品评测 | 想深度评测1个产品 | "Vultr主机详细评测" |
| ⚖️ 产品对比 | 想对比两个产品 | "Vultr vs DigitalOcean对比" |
| 📚 教程指南 | 想教别人怎么用东西 | "如何搭建V2Ray代理" |
| 📦 资源分享 | 想分享付费资源 | "Docker入门课程" |

## 2. 快速入门（5分钟）

### 第 1 步：打开管理系统

在浏览器地址栏输入：
```
https://www.xcodezg.com/zh/admin
```

**注意**：第一次会看到新手引导，跟着一步步来就行！

### 第 2 步：上传图片/视频

1. 点击【媒体库】标签
2. 点击【上传图片/视频】按钮
3. 选择文件（支持 JPG, PNG, GIF, WebP, MP4, WebM）
4. 文件自动上传完成

**更方便的方法**：直接用鼠标把文件拖进去！

### 第 3 步：创建文章

1. 点击【内容管理】标签
2. 点击【+ 创建新内容】按钮
3. 填写信息：
   - **内容类型**：推荐/评测/对比/教程/资源
   - **标题**：中英文标题
   - **封面图**：从媒体库选择
   - **内容区块**：添加文本、图片、视频等
4. 点击"发布"或"保存草稿"

## 3. 内容管理详细指南

### 创建新内容

#### 步骤 1：选择内容类型

根据你的目的选择合适的内容类型。**不知道选哪个？** 选【推荐文章】或【教程指南】最常用！

#### 步骤 2：填写标题

标题示例：
- "2026年最好的5个VPS主机推荐"
- "零基础学会使用ChatGPT"
- "V2Ray完整搭建教程"

**提示**：标题要吸引人，但不要夸张。

#### 步骤 3：填写 SEO 信息

- **SEO 标题**：页面显示的标题（建议 50-60 字符）
- **SEO 描述**：页面描述（建议 150-160 字符）

#### 步骤 4：编辑内容

切换中/英文标签页分别编辑：

**引言框**：写一段简短的介绍
```
如果你正在寻找速度快、价格合适的VPS主机，那就看这篇文章吧！
我们测试了10多个主流VPS服务商，最后选出这5个最好的推荐给你。
```

**正文框**：这是主要的内容，像写 Word 一样

#### 步骤 5：预览和发布

- **保存草稿**：暂存，以后继续编辑
- **立即发布**：文章马上上线，所有人都能看到！

### 编辑已有内容

1. 点击【内容管理】标签
2. 找到要编辑的文章
3. 点击【编辑】按钮
4. 修改后保存或发布

### 删除内容

1. 找到要删除的内容
2. 点击【删除】按钮
3. 确认删除

**注意**：删除后无法恢复，请谨慎操作！

## 4. 媒体库使用

### 上传文件

**支持格式**：
- 图片：JPG, PNG, GIF, WebP
- 视频：MP4, WebM, MOV
- 单文件最大：10MB

### 上传方式

1. **点击上传**：点击按钮选择文件
2. **拖拽上传**：直接把文件拖进去

### 管理媒体

- **筛选**：按类型（图片/视频）筛选
- **搜索**：通过文件名搜索
- **删除**：点击删除按钮删除文件

## 5. SEO 工具

### 自动生成 SEO 数据

1. 进入"SEO 工具"标签页
2. 选择页面类型
3. 输入目标关键词（如：`best vps 2026`）
4. 点击"生成 SEO"
5. 系统自动生成：
   - SEO 标题
   - 页面描述
   - H2 标题建议
   - Schema.org 结构化数据

## 6. 常见问题

### Q1: 上传图片失败？

**检查**：
- 文件是否超过 10MB？
- 格式是否支持？（JPG, PNG, GIF, WebP, MP4, WebM）
- 网络是否正常？

### Q2: 文章保存失败？

**检查**：
- 标题有没有填？
- 内容是否太短？
- 试试【保存草稿】

### Q3: 找不到刚发布的文章？

**解决**：
- 确认点击的是【发布】而不是【保存草稿】
- 刷新网页
- 清理浏览器缓存

### Q4: 想恢复删除的文章？

**答**：当前版本不支持恢复，删除前请三思！

### Q5: 支持的图片格式？

**答**：JPG, PNG, GIF, WebP

## 7. 最佳实践

### ✨ 标题要吸引人

- ❌ 不好："VPS主机推荐"
- ✅ 好："2026年最好的5个VPS主机推荐（附对比图）"

### ✨ 内容要有价值

- 不要写废话
- 提供真正有用的信息
- 读者看完能学到东西

### ✨ 适当使用图片

- 图文并茂，读起来更舒服
- 但不要太多图片，影响加载速度

### ✨ 定期更新内容

- 保持内容新鲜
- 能吸引回头客

### ✨ 注意排版

- 每段话不要太长
- 适当分段
- 用数字列表更清晰
