# 内容管理系统 (CMS) 使用指南

## 📋 系统概述

本网站配备了一个完整的内容管理系统（CMS），让您可以轻松管理网站的所有内容，包括文章、图片、视频等。

---

## 🚀 访问管理后台

### 方式1：直接访问
在浏览器中访问：
```
http://your-domain.com/admin
```

### 方式2：通过部署平台
如果使用 CloudBase、EdgeOne Pages 等平台，请在控制面板中找到管理后台链接。

---

## 📝 内容管理功能

### 1. 创建新内容

1. **进入管理后台**
   - 访问 `/admin` 路径
   - 点击"内容管理"标签页

2. **点击"创建新内容"按钮**

3. **填写内容信息**
   - **内容类型**：选择推荐页、评测页、对比页、教程页或资源页
   - **标题**：填写中英文标题（自动生成URL slug）
   - **URL Slug**：自定义页面URL（如：`best-vps-2026`）
   - **封面图片**：从媒体库选择图片作为封面

4. **配置SEO**
   - **标题**：页面显示的SEO标题（建议50-60字符）
   - **描述**：页面描述（建议150-160字符）

5. **编辑内容**
   - 切换中/英文标签页分别编辑内容
   - **引言**：添加页面开头的引言文字
   - **内容区块**：点击按钮添加不同类型的内容：
     - **文本**：普通文本段落
     - **图片**：插入图片
     - **视频**：插入视频
     - **列表**：创建项目列表

6. **保存或发布**
   - **保存草稿**：保存但不发布，可以继续编辑
   - **发布**：直接发布到网站

### 2. 编辑现有内容

1. 在内容列表中找到要编辑的内容
2. 点击"编辑"按钮
3. 修改内容后点击"保存草稿"或"发布"

### 3. 删除内容

1. 在内容列表中找到要删除的内容
2. 点击"删除"按钮
3. 确认删除操作

### 4. 内容状态

- **草稿**：未发布的内容，访客无法看到
- **已发布**：已上线的内容，所有访客可见
- **归档**：已下线的内容，但保留在系统中

## 🖼️ 媒体库管理

### 上传图片/视频

1. **进入媒体库**
   - 在管理后台点击"媒体库"标签页

2. **上传方式**
   - **点击上传按钮**：选择文件
   - **拖拽上传**：直接拖拽文件到上传区域

3. **支持的格式**
   - **图片**：JPG, PNG, GIF, WebP
   - **视频**：MP4, WebM, MOV

4. **文件限制**
   - 单个文件最大 10MB

### 查看媒体

- **网格视图**：以缩略图形式展示所有媒体文件
- **筛选**：按类型（图片/视频）筛选
- **搜索**：通过文件名或Alt文本搜索

### 管理媒体

- **选择封面**：点击图片选中（用于内容封面）
- **删除媒体**：点击删除按钮删除不需要的文件

## 🔧 SEO工具

### 自动生成SEO数据

1. 进入"SEO工具"标签页
2. 选择页面类型（推荐、评测、对比等）
3. 输入目标关键词（如：`best vps 2026`）
4. 点击"生成SEO"按钮
5. 系统自动生成：
   - SEO标题
   - 页面描述
   - H2标题建议
   - Schema.org结构化数据

### 内部链接生成

- 点击"生成内部链接"按钮
- 系统根据内容关系自动推荐相关内部链接

## 📊 数据分析

### 查看网站数据

1. 进入"数据分析"标签页
2. 查看以下指标：
   - **总浏览量**：网站总访问次数
   - **广告收入**：来自广告的收入
   - **联盟收入**：来自联盟链接的收入
   - **转化率**：访客转化比例

### 热门页面

- 查看表现最好的页面
- 了解每个页面的浏览量和收入

### 优化建议

- 系统会根据数据提供优化建议
- 包括：低转化率提示、高表现页面提示等

## 💡 内容类型说明

### 1. 推荐页 (Recommendation)
- 用于推荐多个产品/服务
- 包含对比表格、详细评测、选择指南
- 最适合变现的页面类型

### 2. 评测页 (Review)
- 深度评测单个产品/服务
- 包含背景、性能、价格、测试结果
- 适合建立信任和权威性

### 3. 对比页 (Comparison)
- 对比两个或多个产品
- 包含对比表格、场景对比、最终推荐
- 适合处于购买决策阶段的用户

### 4. 教程页 (Tutorial)
- 分步教学指南
- 包含前置条件、步骤、常见错误
- 适合吸引有学习需求的用户

### 5. 资源页 (Resource)
- 付费资源或工具
- 包含价值主张、目标人群、定价
- 直接变现方式

## 🎯 内容创建最佳实践

### 1. SEO优化

- **关键词研究**：使用SEO工具生成正确的关键词
- **标题优化**：包含主要关键词，保持简洁吸引人
- **描述优化**：准确描述内容，包含关键词，激发点击欲望
- **结构化数据**：使用Schema.org帮助搜索引擎理解内容

### 2. 内容质量

- **原创内容**：避免抄袭，提供独特价值
- **深度分析**：不仅仅是表面介绍，提供深入见解
- **定期更新**：保持内容时效性
- **多媒体结合**：合理使用图片和视频提升体验

### 3. 转化优化

- **明确的CTA**：每个页面都有明确的行动号召
- **联盟链接**：正确使用联盟链接
- **A/B测试**：测试不同的CTA文字和位置

### 4. 内部链接

- **相关链接**：链接到相关内容，提升用户停留时间
- **锚点优化**：使用描述性锚点文本
- **链接分布**：均匀分布在页面各处

## 🔄 数据备份

### 本地存储（当前）

- 内容存储在内存中（开发模式）
- 服务器重启后数据会丢失
- **建议**：尽快连接真实数据库

### 生产环境建议

使用以下任一数据库服务：

1. **CloudBase（腾讯云）**
   - 完整的云开发平台
   - 数据库、存储、函数一体化
   - 适合微信生态

2. **Supabase**
   - 开源的Firebase替代品
   - PostgreSQL数据库
   - 实时功能

3. **MySQL/MongoDB**
   - 传统数据库
   - 需要自行部署

## 🚀 部署流程

### 步骤1：连接数据库

1. 在项目中注册并连接数据库服务
2. 修改 `src/lib/content-store.ts` 以使用真实数据库
3. 测试连接是否正常

### 步骤2：配置环境变量

创建 `.env.local` 文件：

```env
DATABASE_URL=your-database-connection-string
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

### 步骤3：部署到生产环境

1. 使用集成功能部署到云平台
2. 确保所有API路由正常工作
3. 测试媒体上传功能
4. 验证内容可以正常创建和显示

## ❓ 常见问题

### Q1: 上传的图片在哪里？
A: 图片保存在 `public/uploads/` 目录下，URL为 `/uploads/文件名`

### Q2: 如何批量上传？
A: 当前版本一次上传一个文件，后续版本会支持批量上传

### Q3: 支持哪些图片格式？
A: JPG, PNG, GIF, WebP

### Q4: 如何恢复删除的内容？
A: 当前版本不支持回收站，删除即永久删除，建议定期备份数据库

### Q5: 可以导出内容吗？
A: 可以通过API获取所有内容JSON数据自行导出
